## 🆘 Troubleshooting

### La page affiche un 404?
→ Vérifier que netlify.toml redirect est présent

### Les articles ne chargent pas?
→ Vérifier que les chemins markdown sont corrects
→ Vérifier CORS sur les ressources externes

### Dark mode ne persiste pas?
→ Vérifier que localStorage est autorisé
→ Vérifier la console pour erreurs JS

### Analytics ne track pas?
→ Remplacer `G-XXXXXXX` par votre vraie ID
→ Attendre 24h pour les données à apparaître
